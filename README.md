### What I learned

This was my second frontend mentor challenge and it felt much easier than the first one! I'm already seeing how much i've grown as a beginner.
The biggest things i improved on were:
Semantic HTML- Instead of using div for everything i used more meaningful tags like <article> and <footer> to give my HTML proper structure and meaning.

Box Shadow- I learned how box shadow works and how it values are different from margin and padding. The four values control horizontal offset , vertical offset, blur and color.

Flexbox for alignment- i used flexbox again to align the avatar image and name side by side in the footer.

### Continued development
Going forward i want to keep improving in these areas;

More Semantic HTML- I want to get more comfortable with using other tags like <time>,<figure> and <main>
CSS custom properties-using variables for colors and fonts to keep the card cleaner.
Responsive Design- Making the card look great on all screen sizes.
